# Rapport 3D standaard
In deze repository wordt gewerkt aan  het rapport dat uitkomsten beschrijft van het vervolgonderzoek naar een 3D standaard voor de geo-basisregistraties. 

## ReSpec template instructies

Dit rapport wordt geschreven in ReSpec. ReSpec is een tool om HTML- en PDF-documenten te genereren op basis van markdowncontent. Deze template helpt je bij het opstellen en publiceren van documenten volgens de Geonovum-standaard.

De dynamische voorbeeldpagina van het document is [hier te bekijken](https://geonovum.github.io/3d-standaard-rapport/).

Voor meer over werken met ReSpec zie de [gebruikersinstructie](https://github.com/Geonovum/NL-ReSpec-template?tab=readme-ov-file).
